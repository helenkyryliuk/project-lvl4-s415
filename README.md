## :arrow_forward: How to Run App

1. cd to the repo
2. Setup
  - run `make install`
2. Run Build
  - run `make develop`

## :hammer: Technologies

Project is created with:

* Ajax, REST
* Websockets
* React + Redux + Forms
* Bootstrap on React
* Webpack
